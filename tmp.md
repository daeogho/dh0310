git init : git hub 시작하기위해 폴더나 파일에 접속? || git파일이 생김 -> git 기본세팅이 된다.
git add . : commit 하기위해 내용을 추가하는 것 || 변경사항들 내용을 추가 한다.
git commit -m '' : add한 내용을 저장하기 위한 것 '' 저장내용 이름 || 변경 된 내용에 대해 저장하는 것

git push origin -- : commit한 내용들을 로컬(내컴퓨터)에서 remote(원격) / github 서버로 보내는 것
merge commit : 여러개의 파일? master, dev 등 나누어져 있는 내용들을 합치는 것
git pull origin -- : 원격(git hub)에 있는 것들을 가져 오겠다. /push 반대개념 

git switch -- : master, dev 등 ()안에 변경 폴더개념
git status : 현재 상황 확인
