git init : git hub 시작하기위해 폴더나 파일에 접속?
git add . : commit 하기위해 내용을 추가하는 것
git commit -m '' : add한 내용을 저장하기 위한 것 '' 저장내용 이름

git push origin -- : commit한 내용들을 로컬(내컴퓨터)에서 remote(원격) / github 서버로 보내는 것
merge commit : 여러개의 파일? master, dev 등 나누어져 있는 내용들을 합치는 것
git pull origin -- : push가 되지 않을때 밀어넣는거?

git switch -- : master, dev 등 ()안에 변경 폴더개념
git status : 현재 상황 확인
